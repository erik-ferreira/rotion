## Setup do TailwindCSS

- `npm install -D tailwindcss postcss autoprefixer`
- src > renderer > `npx tailwind init -p`
- electro.vite.config.ts:
  ```ts
  renderer: {
    css: {
      postcss: {
        plugins: [
          tailwindcss({
            config: './src/renderer/tailwind.config.js'
          })
        ]
      }
    },
  }
  ```
